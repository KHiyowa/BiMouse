# BiMouse

## 概要
Windowsのマウス設定（左右ボタンの入れ替え）を、タスクトレイから素早く切り替えるための常駐ツールです。  
右手マウス・左手マウスを頻繁に持ち替える用途向けに製作しました。  

作者はこれまで肥後信嗣氏のフリーソフト「両刀マウス」を使用していましたが、一部ノートPCにタッチパッドの物理クリックボタンが1つしかないものがあり、マウスボタンの左右両押しができず使用に難があったため、アイデアはそのままタスクトレイからも切り替えられるよう再実装したものです。  

## 特徴
- **タスクトレイ常駐**: 現在の状態（右手/左手）がアイコンでひと目で分かります。
- **クイック切り替え**: マウスの左右ボタンを同時に **1秒間** 長押しすると、設定画面を開かずに即座に入れ替わります。
- **セキュリティソフト親和性**: キーロガーと誤検知されやすいフック処理（`SetWindowsHookEx`）を使わないことで、企業導入のセキュリティソフト（FFRI Yarai等）環境下でも誤検知が起きにくくなっています。

## 使い方
1. `BiMouse.exe` を起動します。タスクトレイにアイコンが表示されます。赤色に点灯しているのが主ボタンです。
2. **切り替え方法**:
    - **メニューから**: アイコンをクリック（左/右どちらでも可）し、メニューから「右手マウス」「左手マウス」を選択します。
    - **ショートカット**: マウスの左右ボタンを同時に **1秒間** 長押しします。
3. **終了方法**:
    - タスクトレイアイコンをクリックし、メニューの「終了」を **Ctrlキーを押しながら** クリックします。

## 仕様・制限事項
- **同時押し時の挙動**:
    - 本ツールはOSの入力イベントをフックしない実装のため、左右同時押し判定中も、OSや他のアプリケーションにはクリックイベントが通知されます。
    - そのため、同時押し切り替えを行った際、マウスカーソルの位置によっては右クリックメニューが表示されますが、これは仕様となります。

## 動作環境
- Windows 10 / 11
